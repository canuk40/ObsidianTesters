# ObsidianTesters

A platform where users earn shards by testing apps and spend shards to post their own apps for testing.

## Tech Stack
- Kotlin + Jetpack Compose
- Firebase (Auth, Firestore, Storage, Crashlytics)
- Google Play Billing v7
- Hilt dependency injection
- MVVM architecture
